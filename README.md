사용 설명서
--------------------
원하는 오디오 파일을 원하는 시간에 재생 시켜주는 프로그램입니다.

폴더 내 AUtoDio.exe 파일을 실행하시면 됩니다.

추가 버튼을 눌러 음악 파일과, 재생 될 요일 및 시간을 설정하고 추가하면 해당 시간에 자동으로 음악 파일이 실행됩니다.   
단, 현재(ver 0.2.1-beta1) 추가 가능한 파일 확장자는 *.mp3, *.wav 입니다.
